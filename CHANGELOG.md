# Changelog

## [2.0.0] - 2026-06-10

### Added
- Collapsible item groups in EMI sidebar — group similar items under a single icon, click to expand/collapse
- Per-group border color customization
- Show Group Border / Show Group Fill display options
- Alt + Left Click on a grouped item to collapse its group
- In-game group editor accessible from EMI settings
- Group creation/edit dialog with auto-generated IDs
- Fluid support (water, lava) and tag-based grouping

### Fixed
- Search now works correctly with grouped items
- Drag-and-drop to favorites works with grouped members
- Groups start collapsed by default when joining a world
