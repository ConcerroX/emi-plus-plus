# Changelog

## [2.0.0] - 2026-06-10

- Group items in EMI sidebar — click group icon to expand, Alt+Click member to collapse
- Per-group border color, show/hide border and fill options
- In-game group editor via EMI settings → Edit Groups
- Auto-generated IDs when creating groups (supports Chinese and other scripts)
- Group fluids (water, lava) and tag-based collections
- Search and favorites work correctly with grouped items
